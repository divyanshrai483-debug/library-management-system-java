# Library Management System

## Project Description
A console-based Library Management System developed in Java using Object-Oriented Programming concepts.

## Features
- Add Book
- View Books
- Search Book
- Issue Book
- Return Book
- Save Data to File
- Load Data from File

## Technologies Used
- Java
- OOP (Classes & Objects)
- ArrayList
- File Handling
- Serialization
- Exception Handling

## Project Structure

Book.java
Library.java
Main.java

## How to Run

Compile:

javac *.java

Run:

java Main
